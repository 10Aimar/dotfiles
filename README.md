# DotFiles

Personal dotfiles for my Fedora 44 KDE running nini + noctalia


## Managed with

- GNU Stow
- Git
- GitHub

## Packages

- zsh
- starship
- konsole
- niri
- noctalia

## Installation

Clone the repository:

```bash
git clone https://github.com/10Aimar/dotfiles.git
cd dotfiles
```

Create the symlinks:

```bash
stow zsh
stow starship
stow konsole
stow niri
stow noctalia
```
